Tested up to: 3.2.1
Stable tag: 0.1.3

A plugin which allows you to disable parent menu link created through wp_nav_menu function.

== Description ==

By default WordPress wp_nav_menu function generates a link on parent and child menus.If you want that only child links are clickable then this plugin can solve your problem.

We would be keen you hear your feedback and requests for any enhancements. Please send your feedback to wordpress@mediologysoftware.com.
== Installation ==

1. Upload the `disable-parent-menu-link` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
